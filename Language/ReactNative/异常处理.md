- Android 5.0以下真机调试红屏问题

```js
手机电脑处于同一网段下（电脑使用网线，手机wifi亦可）
打开开发者菜单（摇晃设备或运行`adb shell input keyevent 82`）
`Dev Settings` -> `Debug server host for device` -> 输入电脑IP:8081(例如：10.10.1.1:8081)
Reload JS

**重点：如果上面的方法没有效果，需要检查电脑防火墙是否屏蔽了NodeJS** 
```
